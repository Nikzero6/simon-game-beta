Simon's Game

Rules:

  1. There are 4 buttons.
  2. You will be shown an order in which you have to click the buttons.
  3. Click the buttons in the same order.
  4. Difficulty increases with each level.
  5. Lets see if you can beat the highest score.

Press a key to START the game!
